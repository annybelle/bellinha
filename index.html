calculadora de bhaskara

A fórmula de Bhaskara é utilizada para encontrar as raízes de uma equação quadrática. A fórmula é: x = (-b ± √(b² - 4ac)) / (2a), onde a, b e c são os coeficientes da equação quadrática ax² + bx + c = 0. So ter os valores de a, b e c, para calcular as raízes.

exemplo

equação quadrática: 2x² - 5x + 2 = 0. Neste caso, temos: a = 2 b = -5 c = 2 Substituindo esses valores na fórmula de Bhaskara, temos: x = (-(-5) ± √((-5)² - 4*2*2)) / (2*2) x = (5 ± √(25 - 16)) / 4 x = (5 ± √9) / 4 x1 = (5 + 3) / 4 x1 = 8 / 4 x1 = 2 x2 = (5 - 3) / 4 x2 = 2 / 4 x2 = 0.5 Portanto, as raízes da equação quadrática são x1 = 2 e x2 = 0.5.
